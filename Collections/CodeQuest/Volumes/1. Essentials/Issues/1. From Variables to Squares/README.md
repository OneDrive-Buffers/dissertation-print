# Issue 1: From Variables to Squares

This issue is now split into three layers:

- `guide/` keeps exploratory drafts and generated samples.
- `cover/` keeps the approved cover art.
- `src/` is the production source of truth for the printable issue.

## Production Structure

```text
1. From Variables to Squares/
|-- bibliography/
|-- cover/
|-- guide/
`-- src/
    |-- main.tex
    |-- include/
    `-- sections/
```

## Editorial Direction

The production issue combines:

- the stronger long-form explanations and exercise progression from `guide/codequest_issue1.tex`
- the punchier magazine rhythm and assembly-flavored hints from `guide/pages.tex`

The result is a modular B5 print issue with:

- a full-page cover
- a strong opening hook
- memory, variables, arithmetic, control flow, `while`, `for`, and nested-loop sections
- matrix-pattern exercises for squares, diagonals, triangles, and rhombuses
- a final boss problem for the first issue
